# 商品维护系统

这是一个简单的商品维护系统网页应用。

## 功能

- 单个商品维护
- 批量商品维护

## 如何运行

1. 确保已安装 Python 3.x
2. 在项目目录下运行以下命令：
   ```
   python server.py
   ```
3. 浏览器将自动打开 http://localhost:3068

## 文件结构

- `index.html` - 主页面
- `styles.css` - 样式表
- `app.js` - 交互逻辑
- `server.py` - 本地服务器

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Python 3.x (用于本地服务器)
